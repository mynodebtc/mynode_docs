## What is MyNode?

MyNode is the simplest, most powerful way to run a Bitcoin and Lightning node! We combine the best open source software with our interface, management, and support so you can easily, privately, and securely use Bitcoin and Lightning.

As a self-hosted Bitcoin and Lightning node platform, we have an intuitive browser-based interface and numerous applications, including Mempool, Thunderhub, Electrum Server, BTCPay Server, and many more. These docs cover first-time setup, using the available apps, wallet pairing, remote access, troubleshooting, and advanced configuration options.
